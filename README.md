# Chrono Pièces Gabon (CPG) - Landing Page 🚜🔧

Bienvenue sur le dépôt de la landing page officielle de **Chrono Pièces Gabon (CPG)**, votre partenaire de confiance pour la vente de pièces détachées de véhicules industriels et d'engins lourds au Gabon.

## 🌟 À propos du projet

Ce projet est une **Landing Page statique** conçue pour présenter les catalogues de produits et les services de Chrono Pièces Gabon. L'objectif est d'offrir une interface professionnelle, moderne et rapide pour permettre aux clients (B2B et B2C) de découvrir l'expertise de CPG et de demander des devis facilement.

### 🎯 Fonctionnalités clés
- **Design Industriel Premium** : Thème sombre moderne avec des accents oranges, adapté à l'image du secteur industriel.
- **Catalogue Dynamique** : Présentation claire des catégories de pièces (Auto, Engins Lourds, Outillage, EPI, etc.).
- **Animations Fluides** : Effets de défilement (scroll) et d'apparition pour une expérience utilisateur engageante (via JavaScript Vanilla).
- **Responsive Design** : Interface 100% adaptée aux mobiles, tablettes et ordinateurs de bureau.
- **Appel à l'Action (CTA)** : Intégration directe de boutons vers WhatsApp et Email pour faciliter les demandes de devis rapides.

## 🛠️ Technologies utilisées

- **HTML5** : Structure sémantique de la page.
- **CSS3** : Styles personnalisés (Vanilla CSS), animations, et mise en page Flexbox/Grid (aucun framework lourd n'a été utilisé pour garantir des performances maximales).
- **JavaScript (ES6+)** : Logique d'interaction, animations au défilement, et gestion de la modale de contact.

## 🚀 Installation & Déploiement

Étant donné qu'il s'agit d'un site web statique, aucune installation complexe (comme Node.js) n'est requise.

### Pour visualiser le site localement :
1. Clonez ce dépôt sur votre machine.
2. Ouvrez simplement le fichier `index.html` dans le navigateur web de votre choix (Chrome, Safari, Firefox).

### Pour le mettre en ligne :
Ce projet est optimisé pour être déployé très facilement sur des plateformes d'hébergement gratuites telles que :
- **Netlify** (via Glisser-déposer ou liaison GitHub)
- **Vercel**
- **GitHub Pages**

## 📂 Structure des fichiers
- `index.html` : La page principale contenant tout le contenu.
- `style.css` : La feuille de style gérant l'apparence visuelle.
- `main.js` : Le script gérant les animations et interactions.
- `*.png / *.jpg` : Les ressources visuelles et images utilisées sur le site.
